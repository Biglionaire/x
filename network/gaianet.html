Installation Guide

Install Gaia Node

curl -sSfL 'https://github.com/GaiaNet-AI/gaianet-node/releases/latest/download/install.sh' | bash source ~/.bashrc

Init with the qwen-1.5-0.5b-chat model

gaianet init --config https://raw.githubusercontent.com/GaiaNet-AI/node-configs/main/qwen-1.5-0.5b-chat/config.json

Start the node

gaianet start
